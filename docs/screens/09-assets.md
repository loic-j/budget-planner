## Features

### Chart вҖ” `LineChart (@mui/x-charts)`

- X axis: years from `start_date` to `end_date`
- Y axis: asset value in budget currency
- One line per asset, color-coded by type
- Bold "Total assets" line
- Growth rate visually distinguishable: appreciating assets curve up, depreciating assets curve down and flatten near zero
- Chart selector `[в–ҫ]`: "Individual values", "Stacked total", "Net (assets вҲ’ loans)"
- Hover tooltip: date + value per asset + total

### DataGrid Columns

| Column             | Type   | Notes                                          |
| ------------------ | ------ | ---------------------------------------------- |
| Name               | text   | required                                       |
| Type               | select | REAL_ESTATE / INVESTMENT / VEHICLE / OTHER     |
| Current value      | number | value at acquisition date                      |
| Acquisition date   | date   | projection start for this asset                |
| Annual growth rate | number | % вҖ” positive appreciates, negative depreciates |

Growth rate display: positive shown in green (`+2.5%`), negative in red (`-12%`).

### Asset Value Formula (applied in projection)

```
value(T) = current_value Г— (1 + rate/100) ^ years_since_acquisition
```

Depreciated vehicles floor at вӮ¬0 вҖ” no negative values.

## Proposed Improvements

- **Loan link indicator** вҖ” show a small badge on assets linked to a `LoanDetail`; hovering reveals remaining loan balance and payoff date
- **Revaluation history** вҖ” allow user to log manual revaluations (e.g. after a property appraisal); chart switches from pure formula to piecewise actuals + forecast
- **Asset allocation donut** вҖ” small `PieChart (@mui/x-charts)` alongside the chart showing current % split by asset type (Real estate / Investments / Vehicles / Other)
- **Benchmark overlay** вҖ” toggle to overlay a market index benchmark (e.g. S&P 500 historical average 7%) on investment assets for comparison
